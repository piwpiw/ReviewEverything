# AUTONOMOUS_HOURLY_REPORT

매시간 수행 사이클을 운영 확인용으로 누적한 리포트입니다.
작성/갱신 대상: `autonomous-ops-loop.ts`(`ops:review-list*`, `ops:autonomous*`)

- 주기: 60분 단위
- 출력: `apps/web/docs/AUTONOMOUS_HOURLY_REPORT.md`

## 운영 사용 규칙

- 파일이 존재하고 각 블록이 시간대별로 누적될 때까지는 루프가 실제 동작 중
- 각 블록은 다음을 확인
  - 적용 사이클 범위
  - 실행된 커맨드 분포
  - 누적 검토 항목
- 최종 운영 판단: 5시간 기준으로 누락/중단 여부를 블록 개수(5개)로 점검
